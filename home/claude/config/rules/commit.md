# Commit Rules

## No AI Attribution

Never attribute changes to Claude or any AI assistant in commit messages.

- Do not add `Co-Authored-By: Claude Opus 4.7 (1M context) <noreply@anthropic.com>` or any similar co-author trailer.
- Do not mention that Claude, an AI assistant, or an automated agent made the change.
- Write commit messages as normal project history: what changed and why.
